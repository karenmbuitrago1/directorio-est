## Para empezar

Paso 1: Instalar gulp en el ordenador, para poder visualizar localmente.

```
$ npm install gulp -g
```

### Dependencias

Paso 2: Instalar las dependencias del proyecto.

```
$ npm install
```

```
npm install --save-dev gulp
```

## Compilar

Paso 3: Teniendo todo instalado, ahora debe compilar.

```
$ gulp
```
